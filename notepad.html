Like A Notepad

Productivity • Python • GUI Application

A lightweight and efficient text editor designed for speed and simplicity. Built with Python, it mirrors the classic functionality of standard notepads but with a cleaner codebase.

Features include creating, opening, and saving text files, as well as essential editing tools. It serves as a practical example of building desktop GUI applications.
